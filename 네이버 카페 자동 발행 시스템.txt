네이버 카페 자동 발행 시스템

핵심기능
네이버 카페에 글과 댓글을 다는 시스템 입니다.
1. 카페 특성상 글 작성자, 댓글 작성자(댓글 + 대댓글 + 혹은 그 다음에 댓글까지 모두) 구조로 되어져 있습니다.
2. 네이버 계정별로 ip가 달라야 한다.
3. 네이버 계정별로 ip가 달라야 함으로 pc 여러대에서 ip를 각각 다르게 세팅 후 네이버 로그인 해서 작성해야함
4. pc 여러대가 유동적으로 작업해야함. (작성된 댓글원고 순서에 맞춰서)
5. 실제 사람이 입력하는 것 처럼 한글자씩 입력해야함
6. 네이버는 엄청 나게 봇감지를 하기 때문에 최대한 못인것처럼 보이지 않도록해야하고 파이썬으로 셀리니움을 사용해서 작업하고싶다.
7. 기존 우리 화시 scorp 서버에서 해당 작업을 할 수 있는 페이지를 만들고 싶다.
8. scorp 에서는 작업 현황과 대시보드에서 지금 몇번째 PC에 어떤 네이버 계정이 로그인 되어져 있는지 해당 컴퓨터의 IP 주소가 뭔지 또 필요한 기타 정보들을 보고 실시가 현황을 보여주는 페이지도 만들고싶다. 해당 페이지는 네비게이션 바에 마케팅 관리 우측에 자동화 라는 메뉴를 만들고 거기 하위 목록에 카페라고 만들고 거기를 클릭하면 이동하도록 하고싶다.
9. 만약 첫번째, 두번째 추가 설명에 있는 로직에 의해 작성할 글이 있는데 작업 PC들에서 이미 다른 글을 작성 중이라면 작업 대기 목록에 추가하고 하나씩 차례로 처리하고 싶다. 그리고 작업 대기 목록도 어떻게  대기 중인지 보여 주고 이게 AI가 쓴 글인지 휴먼(사람)이 한 글인지 도 표시해줘, 당연히 상품과 타겟키워드도 표시되야해

추가 설명
1. 글을 발행하는 로직을 2개로 하고싶다
	첫번째(기존 ), 스코프 서버에 사람이 직접 마케팅 관리 - 카페 에서 스케줄에 따라서 글을 작성(모달에서 작성)해서 상태가 완료로 바뀌면 인 경우 해당 글을 해당 카페 해당 계정과 제목, 내용 그리고 각각의 댓글 계정으로 댓글까지 자동으로 작성

	두번째(신규), 두번째는 사람이 글과 댓글을 쓰는게 아니라 AI(클로드) api를 통해서 글과 댓글을 작성하고 그 작성된걸 카페에 위 핵심기능처럼 업로드 하면 됩니다.
	이때 사람이 했을때와 마찬가지로 글 작성 스케줄을 짜야하고 전체 현황 계정 관리 연동 관리 카페 관리 상품 관리가 모두 있어야 한다. 기존 카페 작업 관리 페이지 https://scorp274.com/marketing/cafe 이것과 똑같이 있어야 한다. 다만 여기는 사람이 하는게 아니라AI가 하는 것이기 때문에 작업자 가 없어도 됩니다. 그래서 스케줄 관리에서 자동 생성 시 상품하고 하루에 몇개나 작성할지, 언제부터 언제까지 할지, 입력하고싶고 나머지 로직은 기존 로직과 똑같이 생성하고 싶다. 단 사람이 할땐 주말은 스케줄에서 제외했지만 ai는 주말도 하고싶다. 그리고 상품 관리에서 마케딩 상품 추가는 기존 상품 관리에 있는 상품들을 가져오고 키워드와 연동 관리 계정 관리 카페 관리는 새로운 ai용 db를 만들고 사람과는 다른 db로 작업하고싶다. 그리고 레퍼런스 대신 api 프롬프터를 관리하는 탭이 있어야한다. 여기에 프롬프터로 클로드 api를 이용해서 글과 댓글을 생성하고 카페에 작업해야한다. 그리고 스코프에도 저장해야한다.
