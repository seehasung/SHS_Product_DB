# 📘 AI 자동화 시스템 사용 가이드

**작성일:** 2026-01-11  
**대상:** 시스템 관리자 및 작업자

---

## 🎯 시스템 개요

AI 자동화 시스템은 **네이버 카페/블로그 마케팅을 자동화**하는 시스템입니다.

**핵심 기능:**
- 상품별 맞춤 콘텐츠 자동 생성 (Claude AI)
- 키워드별 차별화된 글 작성
- 8대 Worker PC 자동 배포
- 실시간 모니터링

---

## 📋 사용 절차

### **Step 1: 상품 세팅**

1. **자동화 → 카페 관리** 접속
2. **AI 상품 세팅** 탭 클릭
3. **상품 추가** 버튼 클릭
4. 마케팅 상품 선택 (썸네일 보고 선택)
5. **상세정보 수정** 버튼 클릭
6. **12개 필드** 입력:

```
1. 우리제품명: 실제 판매하는 제품명
2. 우리 제품의 핵심: 제품의 가장 중요한 특징
3. 서브 핵심: 보조 특징
4. 사이즈 & 무게: 실제 스펙
5. 타사 차별점: 경쟁사 대비 장점
6. 유명 브랜드들: 이 카테고리의 유명 브랜드 나열
7. 시장 문제점: 기존 제품들의 문제점
8. 우리 가격: 판매가
9. 시장 평균 가격: 경쟁사 평균가
10. 예상 연령대: 타겟 고객 나이 (예: 20-40대)
11. 예상 성별: 남성/여성/남녀공용
12. 기타 특이사항: (선택사항)
```

7. **마케팅 링크**: 쿠팡/네이버 등 실제 판매 URL
8. **카페/블로그** 체크박스 선택
9. **저장** 버튼 클릭

---

### **Step 2: 키워드 관리**

1. 상품 카드의 **키워드** 버튼 클릭
2. **마케팅에서 동기화** 버튼 클릭
   - 마케팅 상품의 활성 키워드가 추가됨
   - 초기 상태: 모두 **미분류**
3. 각 키워드의 **분류 드롭다운** 선택:
   - **대안성**: "○○ 추천", "○○ 구매" 등 구매 관련
   - **정보성**: "○○ 사용법", "○○ 비교" 등 정보 관련
4. 선택 즉시 자동 저장
5. 필요시 **키워드 추가** 버튼으로 직접 추가

**키워드 분류 예시:**
```
대안성:
- "텀블러 추천"
- "보온병 구매"
- "스텐텀블러 가성비"

정보성:
- "텀블러 세척 방법"
- "보온병 비교"
- "텀블러 사용 후기"
```

---

### **Step 3: 레퍼런스 관리**

1. 상품 카드의 **레퍼런스** 버튼 클릭
2. **마케팅에서 동기화** 버튼 클릭
3. 각 레퍼런스의 **분류 드롭다운** 선택
4. **상세 보기** (눈 아이콘) 클릭하여 전체 내용 확인

---

### **Step 4: 템플릿 작성**

1. **AI 템플릿** 탭 클릭
2. **템플릿 추가** 버튼
3. **템플릿 이름** 입력 (예: "대안성 기본 템플릿")
4. **분류** 선택 (대안성/정보성)
5. **변수 추가:**
   - 드롭박스에서 변수 선택
   - "변수 추가" 버튼 클릭
   - 커서 위치에 `{변수명}` 삽입
6. 저장

**템플릿 예시 (대안성):**
```
제목: {product_name} 추천 - {keyword}

안녕하세요!

오늘은 {product_name}에 대해 이야기해보려고 합니다.

제품 특징:
- {core_value}
- {sub_core_value}

크기: {size_weight}

가격: {our_price} (시장 평균 {market_avg_price})

경쟁 제품 대비 차별점:
{difference}

유명 브랜드: {famous_brands}

시장 문제점:
{market_problem}

추천 대상: {target_age}, {target_gender}

구매 링크: {marketing_link}
```

---

### **Step 5: 프롬프트 생성**

1. **AI 프롬프트** 탭 클릭
2. **프롬프트 추가** 버튼
3. **상품 선택** (드롭박스)
4. **키워드 분류** 선택 (대안성/정보성)
5. **템플릿 선택** (선택사항)
   - 선택한 분류의 템플릿만 표시됨
   - 선택 시 자동으로 입력 필드에 로드
6. **시스템 프롬프트** 입력:
```
당신은 네이버 카페 마케팅 전문가입니다.
자연스럽고 진정성 있는 글을 작성합니다.
광고 티를 내지 않고 실제 사용자처럼 작성합니다.
```

7. **사용자 프롬프트**:
   - 템플릿을 수정하거나
   - 변수 드롭박스로 추가
   - **이미지 생성** 버튼 클릭 (3장 생성 텍스트 자동 추가)
   
8. **Temperature**: 0.7 (창의성)
9. **Max Tokens**: 2000 (글 길이)
10. **저장**

---

### **Step 6: 스케줄 생성**

1. **AI 스케줄** 탭 클릭
2. **스케줄 생성** 버튼
3. **상품 선택**
   - 선택 즉시 **키워드 수** 자동 표시
4. **프롬프트 선택**
   - 해당 상품의 프롬프트만 표시됨
5. **기간 설정** (시작일/종료일)
6. **일일 발행 수** 입력
7. **예상 총 발행 수** 자동 계산 표시
8. **저장**

---

## 💡 사용 팁

### **변수 활용**

**좋은 예:**
```
{product_name}은 {core_value} 때문에 인기가 많습니다.
특히 {target_age} {target_gender} 분들에게 추천합니다.
```

**나쁜 예:**
```
이 제품은 좋습니다. (변수 미사용)
```

### **분류 기준**

**대안성 키워드:**
- 구매 의도가 있는 키워드
- "추천", "구매", "가성비", "인기", "베스트"
- 전환율이 높음

**정보성 키워드:**
- 정보 검색 키워드
- "방법", "비교", "후기", "사용법", "차이"
- 트래픽은 많지만 전환율은 낮음

### **템플릿 작성 팁**

1. **변수는 필수**: 최소 5개 이상 사용
2. **자연스럽게**: 변수를 문장에 자연스럽게 녹여냄
3. **길이 조절**: 너무 짧거나 길지 않게
4. **마케팅 링크**: 글 맨 끝에 자연스럽게 배치

## 🔄 워크플로우

```
1. 마케팅 상품 등록 (기존)
   ↓
2. AI 상품 세팅 (12개 필드 입력)
   ↓
3. 키워드 동기화 & 분류
   ↓
4. 레퍼런스 동기화 & 분류
   ↓
5. 템플릿 작성 (대안성/정보성)
   ↓
6. 프롬프트 생성 (상품별)
   ↓
7. 스케줄 생성
   ↓
8. 자동 발행 시작!
```
