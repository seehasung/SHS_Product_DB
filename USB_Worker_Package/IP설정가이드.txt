IP 설정 빠른 가이드

Windows 10/11:

1. Win + R 키를 누르세요
2. ncpa.cpl 입력 후 Enter
3. 이더넷 연결 우클릭 > 속성
4. Internet Protocol Version 4 선택 > 속성
5. 다음 IP 주소 사용 선택
6. PC 번호에 맞춰 입력:

PC #1: 192.168.1.101
PC #2: 192.168.1.102
PC #3: 192.168.1.103
PC #4: 192.168.1.104
PC #5: 192.168.1.105
PC #6: 192.168.1.106
PC #7: 192.168.1.107
PC #8: 192.168.1.108

서브넷 마스크: 255.255.255.0
기본 게이트웨이: 192.168.1.1
DNS 서버: 8.8.8.8
보조 DNS: 8.8.4.4

7. 확인 > 확인

IP 변경 확인:
cmd 열고 ipconfig 입력
