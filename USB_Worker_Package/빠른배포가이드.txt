🚀 Worker PC 빠른 배포 가이드
================================

📦 처음 설치하는 Worker PC
--------------------------
1. USB_Worker_Package 폴더 전체를 Worker PC에 복사
2. install_worker.py 실행 (자동으로 C:\worker 설치)
3. IP 설정 (IP설정가이드.txt 참조)
4. 바탕화면 아이콘 더블클릭으로 실행!

🔄 이미 설치된 Worker PC
--------------------------
방법 1: 자동 업데이트 (권장)
   - 그냥 프로그램 실행만 하면 됩니다!
   - 서버에서 자동으로 최신 버전 다운로드

방법 2: 수동 업데이트
   - 최신 worker_agent.py만 C:\worker\에 복사
   - 프로그램 재실행

⚠️ 주의사항
--------------------------
- Worker PC에 Python 3.8 이상 필요
- Chrome/Edge 브라우저 필요
- 인터넷 연결 필수

📍 설치 경로
--------------------------
C:\worker\
  ├── worker_agent.py (메인 프로그램)
  ├── worker_backup\ (백업 폴더)
  └── requirements.txt (의존성)

🆘 문제 해결
--------------------------
1. Python 없음: Python 3.8+ 설치
2. 모듈 오류: pip install -r requirements.txt
3. 브라우저 오류: Chrome 재설치
4. 서버 연결 실패: https://scorp274.com 접속 확인
